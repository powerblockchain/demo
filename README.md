### 区块链支付充电桩演示 

> 使用shell脚本（bash），配合TrustNote通用钱包，在树梅派下执行。不依靠任何编程语言。

how to use

如何使用

1. first
第一次输入以下命令。

```
./first.sh
```

this command will get balance or address.txt to save a default balance.

2. checking new payment

之后，执行以下命令即可自动检测有无支付。

use a timer to exec this command
```
./getbalance.sh
```
